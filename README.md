# Foodie – Ordering & Recipe Explorer

Foodie is a full-stack web application that allows users to search, explore, and view recipes in real time using API integration. The project is designed to demonstrate front-end development, API handling, and responsive UI design.

## Project Overview
The application provides an intuitive interface where users can browse recipes, search by keywords, and view detailed recipe information. Data is fetched dynamically from external APIs and displayed in a clean and responsive layout.

## Features
- Real-time recipe search using API integration  
- Responsive and user-friendly interface  
- Dynamic display of recipe details  
- Separate frontend and backend architecture  

## Project Structure
frontend/ – User interface built with HTML, CSS, and JavaScript
backend/ – Server and API handling


## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- API Integration  

## How to Run the Project
1. Clone the repository  
2. Open the `frontend` folder  
3. Run the backend server (if required)  
4. Open `index.html` in a browser  

## Purpose
This project was created to demonstrate practical web development skills, including UI design, API integration, and building interactive web applications.
